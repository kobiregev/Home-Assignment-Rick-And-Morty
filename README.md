# Home-Assignment-Rick-And-Morty

Home assignment using the rick and morty api.

https://rickandmortyapi.com/

## Installation

Install my-project with npm

```bash
  npm install 
```
 ## Runing

Run my-project with npm

```bash
  npm run dev
```
